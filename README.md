# supermarket-sales-dashboard
Dashboard analisis penjualan supermarket berbasis Microsoft Power BI yang menyajikan visualisasi interaktif mengenai revenue, profit, quantity sales, performa produk, dan tren penjualan menggunakan dataset Kaggle yang telah melalui proses data cleaning.

## Tools & Technologies
- Microsoft Power BI Desktop
- Microsoft Excel
- GitHub

## Dataset Source
- Kaggle = Supermarket Sales Dataset (https://www.kaggle.com/datasets/faresashraf1001/supermarket-sales)

## Dataset Information
- Total Data : 1000 transaksi
- Format : Microsoft Excel (.xlsx)
- Periode Data : Januari – Maret 2019

Beberapa atribut yang digunakan antara lain:
- Date
- Branch
- City
- Customer Type
- Gender
- Product Line
- Unit Price
- Quantity
- Payment
- COGS
- Rating

## Dashboard Features
Dashboard menampilkan berbagai visualisasi interaktif, yaitu:
- KPI Total Product
- KPI Quantity Sales
- KPI Revenue
- KPI Net Revenue
- KPI Total Profit
- Profit Trendline
- Revenue by Product Line
- Rating by Product in Avarage
- Total of Paymnet Category
- Detail Revenue (Product line, Tax 5%, Net Revenue, Profit, dan Gross Profit Margin (GPM)

Dashboard juga dilengkapi dengan filter berdasarkan:
- Gender
- Customer Type
- Tahun dan Bulan

## Business Insight
Beberapa insight yang dapat diperoleh dari dashboard ini antara lain:
- Mengetahui total revenue dan profit yang dihasilkan selama periode penjualan.
- Membandingkan performa setiap product line berdasarkan revenue dan rating pelanggan.
- Menganalisis tren profit dari Januari hingga Maret 2019.
- Mengetahui distribusi metode pembayaran yang paling banyak digunakan pelanggan.
- Membantu manajemen dalam mengevaluasi strategi penjualan berdasarkan data historis.
